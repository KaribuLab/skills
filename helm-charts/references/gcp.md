# GCP Kubernetes Engine

GCP Kubernetes Engine is a managed Kubernetes service provided by Google Cloud Platform. All helm charts for GCP Kubernetes Engine should follow the best practices for Kubernetes deployment.

## Features

- Managed Kubernetes service.
- Automatic scaling.
- Automatic updates.
- Automatic backups.
- Automatic logging.
- Automatic monitoring.
- Automatic alerting.
- Automatic scaling.
- Use of service accounts for authentication.
